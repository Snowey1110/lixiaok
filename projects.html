Timeline — Work & Projects

Filter by type, then narrow by tag. New rows start as a short summary and dates; you can add images, links, and tags any time in js/timeline-data.js.

How to add a new line on the timeline

Open js/timeline-data.js and push another object into TIMELINE_EXPERIENCES. Set type to work, project, or game, write summary and dateStart / dateEnd, and use draft: true until you add a preview and links. You can also describe what you need in chat and the entry can be filled in here for you.
